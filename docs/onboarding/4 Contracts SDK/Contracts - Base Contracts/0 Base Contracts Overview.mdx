# Base Contracts

<div className="heroGrid">
  
  <div>
    <p>
      Each base smart contract packages together a unique set of smart contract <a href='/contractkit/features'>features</a> so that you can focus on
      adding your custom logic on top, instead of starting from scratch.
    </p>

Rather than implementing EIP standards manually, base contracts provide a logical foundation for you to extend and customize to your needs.

We provide several base contracts for [ERC721](/contractkit/base-contracts/erc-721/erc721base), [ERC1155](/contractkit/base-contracts/erc-1155/erc1155base), and
[ERC20](/contractkit/base-contracts/erc-20/erc20base); each with different levels of functionality depending on your use case.

You can override and add additional functionality to the base logic, and include [features](/contractkit/features) to supplement the base functionality.

</div>
