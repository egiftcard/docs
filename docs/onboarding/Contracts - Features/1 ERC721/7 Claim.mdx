# ERC721Claimable

ERC721Claimable allows you to set up **claim conditions** that define the criteria of how the NFTs that you have lazy
minted (using [ERC721Droppable](/extensions/erc-721/droppable)) can be claimed.

## Unlocked Features

By implementing `ERC721Claimable`, you unlock the following features in the SDK and dashboard:

| SDK Usage                                                                                             | Description                                                                       |
| ----------------------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------- |
| [Claim Conditions](/sdk/interacting-with-contracts/erc721/ERC721Claimable#configure-claim-conditions) | Mints a quantity of lazy-minted NFTs from the drop to a specified wallet address. |

import "@thirdweb-dev/contracts/extension/LazyMint.sol";
import "@thirdweb-dev/contracts/extension/DropSinglePhase.sol";

    constructor(
        string memory _name,

    ) internal virtual override {

    }

    function transferTokensOnClaim(address _to, uint256 _quantityBeingClaimed)
        internal
        virtual
        override
        returns (uint256 startTokenId)
    {
        // Your custom implementation here
    }
}
