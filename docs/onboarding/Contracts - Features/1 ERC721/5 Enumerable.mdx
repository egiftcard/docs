## Unlocked Features

By implementing `ERC721Enumerable`, you unlock the following features in the SDK and dashboard:

| SDK Usage                                                                                 | Description                                                                         |
| ----------------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------- |
| [View Owned NFTs](/sdk/interacting-with-contracts/erc721/ERC721Enumerable#get-owned-nfts) | View all the NFTs a wallet address owns from this collection at this point in time. |
