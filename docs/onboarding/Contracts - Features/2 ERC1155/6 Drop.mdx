# ERC1155Droppable

ERC1155Droppable allows you to lazy-mint a batch of NFTs that can be claimed by other wallet addresses in the future.

:::tip ERC1155Drop Starter Kit
For a fully implemented ERC1155Drop with lazy minting, delayed reveals, and claiming configured,
check out the [ERC1155Drop Starter Kit](/extensions/base-contracts/erc-1155/erc1155drop).
:::

## Unlocked Features

By implementing the ERC1155Mintable contract, you unlock the following features in the SDK and dashboard:

| SDK Usage                                                                                                                     | Description                                                                   |
| ----------------------------------------------------------------------------------------------------------------------------- | ----------------------------------------------------------------------------- |
| [Lazy Mint Batch](/sdk/interacting-with-contracts/erc721/erc1155droppable#create-a-batch-of-nfts-to-be-claimed-in-the-future) | Pass in an array of NFT metadata and lazy-mint them for others to claim.      |
| [Delayed Reveals](/sdk/interacting-with-contracts/erc721/erc721droppable#delayed-reveal)                                      | Pass in an array of NFT metadata to encrypt it and reveal it at a later time. |
