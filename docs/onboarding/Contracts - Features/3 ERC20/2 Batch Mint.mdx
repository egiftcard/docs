# ERC20BatchMintable

ERC20 is the standard for representing fungible tokens; where each token is of equal valuable and interchangeable.

## Base Contracts Implementing This Feature

No base contracts implement this functionality by default.

## Unlocked Features

By implementing the `ERC20` standard, you unlock the following features in the SDK and dashboard:

| SDK Feature                                                                                                | Description                                          |
| ---------------------------------------------------------------------------------------------------------- | ---------------------------------------------------- |
| [Batch Mint Tokens](/sdk/interacting-with-contracts/erc721/erc20batchmintable#mint-tokens-to-many-wallets) | Mint tokens to many wallets in a single transaction. |

    constructor(
        string memory _name,
        string memory _symbol

            _symbol

}
