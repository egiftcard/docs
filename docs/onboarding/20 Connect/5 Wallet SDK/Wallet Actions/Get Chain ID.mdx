## useChainId

Hook for retrieving the chain ID of the network the user is connected to.

```jsx
import { useChainId } from "@thirdweb-dev/react";

const chainId = useChainId();
```

### Usage

```jsx
import { useChainId } from "@thirdweb-dev/react";

function App() {
  const chainId = useChainId();

  return <div>{chainId}</div>;
}
```

### Return value

<details>
  <summary>Return Value</summary>
  <div>

A `number` representing the current chain id, or `undefined` if the user is not connected to a wallet.

```ts
number | undefined;
```

For Example, if the user is connected to the Ethereum Mainnet, the return value will be `1`.

## GetChainId

Get the chain ID of the currently connected wallet.

### Usage

Call this method to get the chain ID of the connected wallet.

```csharp
var data = await sdk.wallet.GetChainId();
```

### Return Value

Returns the chain ID as an `int`.

```csharp
int
```
