## useDisconnect

Hook for disconnecting the currently connected wallet.

```jsx
import { useDisconnect } from "@thirdweb-dev/react";
```

### Usage

```jsx
import { useDisconnect } from "@thirdweb-dev/react";

function App() {
  const disconnect = useDisconnect();

  return <button onClick={disconnect}>Disconnect</button>;
}
```
