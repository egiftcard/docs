## What is the `tx_overrides` parameter?

`tx_overrides` is an optional parameter in the body of certain endpoints to override the gas values & `value` field of a blockchain transaction.

## How do I use my own RPC URLs?

Set the `CHAIN_OVERRIDES` environment variable to the path of the JSON file or publicly accessible URL containing the RPC URLs. The default value is `./chain-overrides.json`. See <a href="https://github.com/thirdweb-dev/engine/blob/main/chain-overrides.example.json" target="_blank">chain-overrides.example.json</a>.

## Why can't Engine connect to my Postgres DB?

Here are some common troubleshooting tips:

- Ensure the Postgres DB is running.
  - If DB running on Docker:
    - In same container, ensure the `host` in the `POSTGRES_CONNECTION_URL` is set to `localhost`.
    - In different container, ensure the `host` in the `POSTGRES_CONNECTION_URL` is set to `host.docker.internal`.
- Ensure the Postgres DB connection URL and credentials are correct.
- Ensure the database name exists on the Postgres DB.
