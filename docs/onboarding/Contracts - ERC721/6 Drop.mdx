# ERC721Droppable

ERC721Droppable allows you to lazy-mint a batch of NFTs that can be claimed by other wallet addresses in the future.

:::tip ERC721Drop Starter Kit
For a fully implemented ERC721Drop with lazy minting, delayed reveals, and claiming configured,
check out the [ERC721Drop Starter Kit](/extensions/base-contracts/erc-721/erc721drop).
:::

## Unlocked Features

By implementing the ERC721Mintable contract, you unlock the following features in the SDK and dashboard:

| SDK Usage                                                                                                                           | Description                                                                   |
| ----------------------------------------------------------------------------------------------------------------------------------- | ----------------------------------------------------------------------------- |
| [Lazy Mint Batch](/sdk/interacting-with-contracts/erc721/ERC721Droppable#create-a-batch-of-unique-nfts-to-be-claimed-in-the-future) | Pass in an array of NFT metadata and lazy-mint them for others to claim.      |
| [Delayed Reveals](/sdk/interacting-with-contracts/erc721/erc721droppable#delayed-reveal)                                            | Pass in an array of NFT metadata to encrypt it and reveal it at a later time. |
