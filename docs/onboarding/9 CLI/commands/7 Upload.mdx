# Upload

Upload any file or directory to decentralized storage ([IPFS](/glossary/ipfs)).

[Learn more about Storage](/storage).

```bash
npx thirdweb upload <path/to/file.png>
```

## Usage

```bash
npx thirdweb upload [options] [path]
```
