# Build

Compile your smart contracts and detect any [extensions](/solidity/extensions) you have implemented.

```bash
npx thirdweb build
```

## Usage

This command:

- Detects the framework you're using (Hardhat, Forge, etc.)
- Compiles all of your smart contracts using the framework's compiler, or the default Solidity compiler
- Displays the [extensions](/solidity/extensions) you have implemented for each contract

```bash
npx thirdweb [options]
```
