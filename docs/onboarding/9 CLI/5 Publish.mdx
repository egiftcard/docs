# Publish

[Publishes](/publish) a version of your protocol so other devs can deploy them and unlock [SDKs](/sdk), [Dashboards](/dashboard) and Analytics. This command:

- Compiles your smart contracts
- Uploads the contract [ABI](https://docs.soliditylang.org/en/v0.8.17/abi-spec.html)s to IPFS
- Allows you to [publish](/publish) them through the dashboard

## Command

```bash
npx thirdweb publish [options]
```
