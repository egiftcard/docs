## Get Started

Set up a new web application with the [React](/react) and [TypeScript](/typescript) SDKs set up using the CLI:

```bash
npx thirdweb create --template next-typescript-solana-starter
```

If this is your first time building on Solana, use our guide to help you get up and running:

### Manually Setup the SDK

Alternatively, install the required packages to start building your app with:

```bash npm2yarn
npm install @thirdweb-dev/sdk @thirdweb-dev/react @solana/wallet-adapter-wallets @solana/wallet-adapter-react
```

And follow one of our guides to configure the SDK:
