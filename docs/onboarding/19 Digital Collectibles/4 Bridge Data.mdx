# Bridge Web2 and Web3 Customer Data

thirdweb’s [Auth SDK](https://portal.thirdweb.com/auth/how-auth-works) is a powerful framework for incorporating a user’s on-chain identity (wallet address) with their off-chain profile. Once authenticated, this information can be integrated into existing application profile systems in a way that existing merchant data can be connected to digital assets owned on the blockchain by the user.
