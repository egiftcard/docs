# ERC721 Drop

The `ERC721Drop` smart contract implements the [ERC721](https://eips.ethereum.org/EIPS/eip-721) NFT standard, along with the [ERC721A](https://www.erc721a.org/) optimization to the standard. It also uses `LazyMint`, `DelayedReveal` and `DropSinglePhase` contract extension.

The `drop` mechanism in the `DropSinglePhase` extension is a distribution mechanism for lazy minted tokens. It lets you set restrictions such as a price to charge, an allowlist etc. when an address attempts to mint lazy minted tokens.

The `ERC721Drop` contract lets you lazy mint tokens, and distribute those lazy minted tokens via the drop mechanism.

## Availability functionality

| Functionality                                                                    | Description                                                                                                                                                                       |
| -------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| [ERC721A](https://github.com/chiru-labs/ERC721A/blob/main/contracts/ERC721A.sol) | Gas-optimized NFT Collection Base Contract.                                                                                                                                       |
| Multicall                                                                        | Capability for better developer experience for your contract; the ability to batch together multiple contract calls in a single call.                                             |
| [Ownable](/extensions/features/ownable)                                          | Ownership of the contract for the contract deployer, and (1) the ability to transfer this ownership, and (2) restrict functions to be callable only by the owner of the contract. |
| Burnable                                                                         | Ability for the owner of a token to transfer their NFT to a null address (cannot be recovered).                                                                                   |
| [Primary Sale](/extensions/features/primarysale)                                 | Accept a price in the currency of your choice from wallets minting your collection’s NFTs.                                                                                        |
| [Royalty](/extensions/features/royalty)                                          | Full royalty support on marketplaces like OpenSea, and EIP 2981 NFT Royalty Standard compliance.                                                                                  |
| [ContractMetadata](/extensions/features/contractmetadata)                        | Set a metadata URI on the contract, to give your collection a name, image, description and symbol.                                                                                |
| [LazyMint](/extensions/features/lazymint)                                        | Prepare the metadata for NFTs that will be minted by an external party, without paying the gas cost for actually minting the NFTs.                                                |
| [DelayedReveal](/extensions/features/delayedreveal)                              | Distribute NFTs to your audience and reveal the metadata of the distributed NFTs, after the fact.                                                                                 |
| [DropSinglePhase](/extensions/features/drop)                                     | Set restrictions such as a price to charge, an allowlist etc. when an address attempts to mint lazy minted tokens.                                                                |

## Implementing the Contract

Import the contract extension and make your contract inherit it.

```solidity
// SPDX-License-Identifier: Apache-2.0
pragma solidity ^0.8.0;

import "@thirdweb-dev/contracts/base/ERC721Drop.sol";

contract MyNFT is ERC721Drop {
    constructor(
        string memory _name,
        string memory _symbol,
        address _royaltyRecipient,
        uint128 _royaltyBps,
        address _primarySaleRecipient
    )
        ERC721Drop(
            _name,
            _symbol,
            _royaltyRecipient,
            _royaltyBps,
            _primarySaleRecipient
        )
    {}
}
```
