# Templates

Kick start your projects using one of our starter kits or templates by using the [thirdweb CLI](https://github.com/thirdweb-dev/js/tree/main/packages/cli):

<!-- Uses JSX cuz it has better syntax highlighting -->

```jsx
npx thirdweb create --template <template-name>
```
