# Contract Settings

You can change the settings of your contract using the SDK, including:

- Contract Metadata
- Royalty Fees
- Platform Fees

## Contract Metadata

You can directly read, write, and update the metadata of a smart contract using the SDK, such as the:

- Image
- Name
- Description

### Read

<ThirdwebCodeSnippet
  contract={"ContractMetadata"}
  name={"get"}
  isFeatureSnippet={true}
/>

### Write (Overwrite)

:::warning
This will **overwrite** the existing metadata.
:::

<ThirdwebCodeSnippet
  contract={"ContractMetadata"}
  name={"set"}
  isFeatureSnippet={true}
/>

### Update

<ThirdwebCodeSnippet
  contract={"ContractMetadata"}
  name={"update"}
  isFeatureSnippet={true}
  isGetContractCode={true}
/>
