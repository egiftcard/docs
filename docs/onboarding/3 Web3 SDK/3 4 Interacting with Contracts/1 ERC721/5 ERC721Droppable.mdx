# ERC721 - Lazy Mint & Claim NFTs

You can utilise these features of the SDK on your contract if it implements [ERC721Droppable](/extensions/erc-721/lazy-mint).

### Create a batch of unique NFTs to be claimed in the future

Create batch allows you to create a batch of many unique NFTs in one transaction.
