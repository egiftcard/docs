# ERC1155 - Claim Conditions

ERC1155Claimable is an extension of [ERC1155Droppable](/extensions/erc-1155/drop) that enables claim conditions.

You can utilize these features of the SDK on your contract if it implements [ERC1155Claimable](/extensions/erc-1155/claim).

## Claim A Lazy-Minted NFT

Claim the next available NFT(s) from the lazy minted batch.

Unless otherwise specified in your contract, the default is to claim the next available token IDs; starting from `0`.

## Configure claim conditions

Define the criteria of how NFTs can be claimed from your drop.

Each claim phase can define:

- When the claim phase will start
- How many NFTs to drop
- How much to charge per NFT
- What currency to use to sell the NFTs
- Which wallet addresses are allowed to mint (allowlist)
- How many NFTs can be claimed per transaction
- How many seconds do wallets have to wait between claims
