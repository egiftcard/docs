# ERC1155Enumerable



You can utilise these features of the SDK on your contract if it implements [ERC1155Enumerable](/extensions/erc-1155/erc1155enumerable).

### Get All NFTs





Get all the data associated with every NFT in this contract.

By default, returns the first 100 NFTs, use queryParams to fetch more.



<CodeSnippetWithSummary
      contract={`ERC1155Enumerable`}
      name={`all`}
      isFeatureSnippet={true}
      isGetContractCode={false}
      showHr={true}
      dontShowSummary
    />
  

### Get Owned NFTs





Get all the data associated with the NFTs owned by a specific wallet.



<CodeSnippetWithSummary
      contract={`ERC1155Enumerable`}
      name={`owned`}
      isFeatureSnippet={true}
      isGetContractCode={false}
      showHr={false}
      dontShowSummary
    />
