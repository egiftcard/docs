# Contract Settings

Configure the settings of your smart contract.

## Contract Metadata

Read, write, and update the metadata of a smart contract using the SDK,
such as the name, description and image.

### Read

<ThirdwebCodeSnippet
  contract={"ContractMetadata"}
  name={"get"}
  isFeatureSnippet={true}
/>

### Write (Overwrite)

:::warning
This will **overwrite** the existing metadata.
:::

<ThirdwebCodeSnippet
  contract={"ContractMetadata"}
  name={"set"}
  isFeatureSnippet={true}
/>

### Update

<ThirdwebCodeSnippet
  contract={"ContractMetadata"}
  name={"update"}
  isFeatureSnippet={true}
  isGetContractCode={true}
/>
