# ERC721 - Claim

ERC721Claimable is an extension of [ERC721LazyMintable](/extensions/erc721lazymintable) that allows you to claim lazy minted NFTs

You can utilize these features of the SDK on your contract if it implements [ERC721Claimable](/extensions/erc721claimable).

## Claim A Lazy-Minted NFT

Claim the next available NFT(s) from the lazy minted batch.

Unless otherwise specified in your contract, the default is to claim the next available token IDs; starting from `0`.
