# ERC1155 - Claim

ERC1155Claimable is an extension of [ERC1155LazyMint](/sdk/interacting-with-contracts/erc721/erc1155lazymintable) that allows users to claim lazy minted NFTs.

You can utilize these features of the SDK on your contract if it implements [ERC1155Claimable](/extensions/erc-1155/claim).

## Claim A Lazy-Minted NFT

Claim the next available NFT(s) from the lazy minted batch.

Unless otherwise specified in your contract, the default is to claim the next available token IDs; starting from `0`.
