The following wallets are supported by the React SDK:

- Injected Wallet (Browser default wallet)
- Coinbase Wallet (Only on WebGL)
- MetaMask Wallet
- WalletConnect
- Magic Link (not on Desktop)
- Paper Wallet
- Local Wallet
- Smart Wallet
- HyperPlay Wallet
