## useAddress

Hook for getting the currently connected wallet address.

Returns `undefined` if no wallet is connected.

```jsx
import { useAddress } from "@thirdweb-dev/react";

const address = useAddress();
```

### Usage

Call the hook to get the current wallet address.

```jsx
import { useAddress } from "@thirdweb-dev/react";

function App() {
  const address = useAddress();

  if (!address) return <div>No wallet connected</div>;

  return <div>My wallet address is {address}</div>;
}
```

### Return value

<details>
  <summary>Return Value</summary>
  <div>

```ts
string | undefined;
```

- Returns a `string` containing the wallet address.
- Returns `undefined` if no wallet is connected.

## GetAddress

Get the address of the currently connected wallet.

### Usage

Call this method to get the address of the connected wallet.

```csharp
var data = await sdk.wallet.GetAddress();
```

### Return Value

Returns the wallet address as a string.

