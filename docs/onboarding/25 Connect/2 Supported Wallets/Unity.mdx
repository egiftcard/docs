The following wallets are supported and appear by default in the Connect Wallet component in the Unity SDK:

- Injected Wallet (Browser default wallet)
- Coinbase Wallet (Only on WebGL)
- MetaMask Wallet
- WalletConnect
- Magic Link (not on Desktop)
- Paper Wallet
- Local Wallet
- Smart Wallet
- HyperPlay Wallet
