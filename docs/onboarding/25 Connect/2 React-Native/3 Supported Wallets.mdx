The following wallets are supported by the React Native SDK:

- [MetaMask](/react-native/wallets#metamask-rainbow-and-trust-wallets) (default)
- [Coinbase Wallet](/react-native/wallets#coinbase-wallet) (default)
- [WalletConnect](/react/react.walletconnectwallet) (default)
- [Trust Wallet](/react-native/wallets#metamask-rainbow-and-trust-wallets) (default)
- [Rainbow Wallet](/react-native/wallets#metamask-rainbow-and-trust-wallets) (default)
- [Smart Wallet](/react-native/wallets#smart-wallet)
- [Paper Wallet](/react-native/wallets#emailphone-number-wallet)
- [Local Wallet](/react-native/wallets#local-wallet)
- [Magic Link](https://github.com/thirdweb-dev/js/blob/main/packages/react-native/src/evm/wallets/wallets/magic-link.tsx)
