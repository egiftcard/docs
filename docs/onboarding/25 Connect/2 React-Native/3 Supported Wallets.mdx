title: Supported Wallets
---

The following wallets are supported by the React SDK:

- [MetaMask](/react/react.metamaskwallet) (default)
- [Coinbase Wallet](/react/react.coinbasewallet) (default)
- [WalletConnect](/react/react.walletconnectwallet) (default)
- [Trust Wallet](/react/react.trustwallet) (default)
- [Rainbow Wallet](/react/react.rainbowwallet) (default)
- [Smart Wallet](react/react.smartwallet)
- [Paper Wallet](/react/react.paperwallet)
- [Local Wallet](/react/react.localwallet)
