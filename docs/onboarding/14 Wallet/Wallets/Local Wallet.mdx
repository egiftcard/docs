Allow users to connect to your app by generating a [Local Wallet](/glossary/local-wallet) directly in your application.

## Usage

```javascript
import { LocalWallet } from "@thirdweb-dev/wallets";

const wallet = new LocalWallet();

wallet.generate();
```

## Configuration

Optionally, provide a configuration object when instantiating the `LocalWallet` class.

<details>
  <summary>chains</summary>
  <div>

Provide an array of chains you want to support.

Must be an array of `Chain` objects, from the [`@thirdweb-dev/chains`](https://www.npmjs.com/package/@thirdweb-dev/chains) package.

Defaults to our [default chains](/react/react.thirdwebprovider#default-chains).

```javascript
import { LocalWallet } from "@thirdweb-dev/wallets";
import { Ethereum } from "@thirdweb-dev/chains";

const walletWithOptions = new LocalWallet(
  // highlight-start
  {
    chains: [Ethereum],
  },
  // highlight-end
);
```

</div>
</details>

<details>
  <summary>dappMetadata</summary>
  <div>

Information about your app that the wallet will display when your app tries to connect to it.

Must be an object containing `name`, `url`, `description` and `logoUrl` properties.

```javascript
import { LocalWallet } from "@thirdweb-dev/wallets";

const walletWithOptions = new LocalWallet({
  // highlight-start
  dappMetadata: {
    name: "thirdweb powered dApp",
    url: "https://thirdweb.com",
    description: "thirdweb powered dApp",
    logoUrl: "https://thirdweb.com/favicon.ico",
  },
  // highlight-end
});
```

</div>
</details>

<details>
  <summary>storage</summary>
  <div>

This is the default storage for storing the private key, mnemonic or encrypted JSON. This can be implemented in any way you want, as long as it conforms to the `AsyncStorage` interface:

```typescript
export interface AsyncStorage {
  getItem(key: string): Promise<string | null>;
  setItem(key: string, value: string): Promise<void>;
  removeItem(key: string): Promise<void>;
}
```

Example:

```javascript
import { LocalWallet } from "@thirdweb-dev/wallets";

const walletWithOptions = new LocalWallet(
  // highlight-start
  {
    storage: {
      getItem: (key) => {
        // Implement your own storage logic here
      },
      removeItem: (key) => {
        // Implement your own storage logic here
      },
      setItem: (key, value) => {
        // Implement your own storage logic here
      },
    },
  },
  // highlight-end
);
```

  </div>
</details>

## Methods

Once instantiated, utilize the following methods to interact with the Local Wallet.

### loadOrCreate

Load the saved wallet data from storage, if it exists, or generate a new one and save it

```javascript
password = "my-password"
wallet.loadOrCreate({
    strategy: "encryptedJson",
    password: password,
});
```

<details>
  <summary>Configuration</summary>
  <div>

#### options

Must be a `LoadOptions` object:

```typescript
type LoadOptions = {
    strategy: "encryptedJson";
    password: string;
    storage?: AsyncStorage;
} | {
    strategy: "privateKey";
    storage?: AsyncStorage;
    encryption: DecryptOptions;
} | {
    strategy: "mnemonic";
    storage?: AsyncStorage;
    encryption: DecryptOptions;
};
```

Example:

```javascript
wallet.loadOrCreate({
    // highlight-start
    strategy: "encryptedJson",
    password: "password",
    // highlight-end
});
```

</div>
</details>

### generate

Creates a new random wallet.

```javascript
const address = await wallet.generate();
```

<details>
  <summary>Configuration</summary>
  <div>

#### Return Value

Returns a `Promise` which resolves to a `string` containing the wallet address.

```typescript
Promise<string>;
```

</div>
</details>

### import

Create a local wallet from an "encryptedJson", "privateKey" or "mnemonic"

```javascript
const address = await wallet.import({
    encryptedJson: "encrypted-json",
    password: "my-password",
});
```

<details>
  <summary>Configuration</summary>
  <div>

#### options


Must be a `ImportOptions` object:


```typescript
type ImportOptions = {
    privateKey: string;
    encryption: DecryptOptions;
} | {
    mnemonic: string;
    encryption: DecryptOptions;
} | {
    encryptedJson: string;
    password: string;
};
```

Example:

```javascript
wallet.import({
    // highlight-start
    strategy: "encryptedJson",
    password: "password",
    // highlight-end
});
```

#### Return Value

Returns a `Promise` which resolves to a `string` containing the wallet address.

```typescript
Promise<string>;
```

</div>
</details>

### isSaved

Get an array of the events for which the emitter has registered listeners.

```javascript
const isSaved = wallet.isSaved();
```

<details>
<summary>Configuration</summary>
<div>

#### Return Value

Returns a `Promise` which resolves to a `boolean` - `true` if wallet data is saved in storage, `false` otherwise.

```typescript
Promise<boolean>;
```

</div>
</details>

### deleteSaved

Delete the saved wallet from storage.

```javascript
await wallet.deleteSaved();
```

### export

Encrypts the wallet with given password and returns the encrypted wallet

```javascript
await wallet.export({
    strategy: "encryptedJson",
    password: "password",
});
```

  <summary>Configuration</summary>
  <div>

```typescript
    password: string;
} | {
    strategy: "privateKey";
    encryption: EncryptOptions;
} | {
    strategy: "mnemonic";
    encryption: EncryptOptions;
};
```

Example:

```javascript
wallet.export({
    // highlight-start
```

### getSavedData

Get the saved wallet data from storage.

```javascript
const wallet = wallet.getSaved( storage: {
      getItem: (key) => {
        // Implement your own storage logic here
      },
      removeItem: (key) => {
        // Implement your own storage logic here
      },
      setItem: (key, value) => {
        // Implement your own storage logic here
      },
    },);
```

<details>
  <summary>Configuration</summary>
  <div>

#### storage

Must be an `AsyncStorage` object.

```typescript
export interface AsyncStorage {
    getItem(key: string): Promise<string | null>;
    setItem(key: string, value: string): Promise<void>;
    removeItem(key: string): Promise<void>;
}
```

Example:

```javascript
wallet.getSaved({
    // highlight-start
    getItem: (key) => {
        // Implement your own storage logic here
    },

        // Implement your own storage logic here
    },

        // Implement your own storage logic here
```

#### Return Value

```typescript
Promise<WalletData | null>
```
