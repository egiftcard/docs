# Unity

The [Unity SDK](/unity) is a set of easy-to-use cross-platform APIs and Prefabs that enable games developers to build seamless web3 experiences to enhance and extend player experiences. <b>Focus on building your game, not on blockchain infrastructure.</b>

## Core Features

The Unity SDK provides a set of core features that enable developers to build web3 experiences into their games. These include:

- Invisible embedded wallets
- Seamless login with email
- Gasless and signless transactions via relayers or account abstraction
- External wallet integrations
- Extremely easy to integrate and customize through the use of prefabs
- Supports interactions with any contract
- Built-in extensions for popular interfaces
- Unified API across multiple platforms, and hundreds of blockchains
