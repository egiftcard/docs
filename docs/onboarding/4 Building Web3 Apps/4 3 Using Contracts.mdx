The SDK can read `views`, `mappings`, and call `functions` on your smart contract from the connected wallet.

You can view all of the available functions on your contract and how to use them from the [dashboard](https://thirdweb.com/dashboard).

## Read Contract Functions

Use the name of the function in the contract along with any required arguments to read data from your smart contract.

<!-- TODO: Snippets component rather than this manual stuff here -->

**React:**

```jsx
const { data, isLoading, error } = useContractData(
  contract, // contract instance
  "functionName", // The name of the function on the contract
  ...args, // The arguments to pass to the function
);
```

**TypeScript**:

```javascript
const result = await contract.call("functionName");
```

## Write Transactions

Use the [useContractData](/react/react.usecontractcall#usecontractcall-function) hook
or the TypeScript SDK's [call](https://portal.thirdweb.com/typescript/sdk.thirdwebsdk.getcontract) method
to call a contract from the connected wallet by using the function's name.

<!-- TODO: Snippets component rather than this manual stuff here -->

**React:**

```jsx
const { contract } = useContract("{{contract_address}}");
const {
  mutate: myFunction,
  isLoading,
  error,
} = useContractCall(contract, "myFunction");
```

**TypeScript:**

```javascript
const result = await contract.call("functionName", argumentName);
```
