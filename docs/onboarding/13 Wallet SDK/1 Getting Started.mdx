## Connecting to a Wallet

To connect to a wallet you need to (1) create an instance of the wallet (2) connect to the wallet e.g. for [Coinbase Wallet](/wallet-sdk/coinbase-wallet):

```typescript
const coinbaseWallet = new CoinbaseWallet();
coinbaseWallet.connect()
```

### Default Setup

<br />

:::note 
The default wallet connectors are [Metamask](/wallet-sdk/metamask), [Coisbase Wallet](/wallet-sdk/coinbase-wallet) and [WalletConnect](/wallet-sdk/walletconnect)
:::

<br/>

Wrap your application in the [`ThirdwebProvider`](/react/react.thirdwebprovider) component to start using the SDK.
Without providing any wallets to the [`ThirdwebProvider`](/react/react.thirdwebprovider), the default wallets will be used. 

```jsx
import { ThirdwebProvider } from "@thirdweb-dev/react";

const App = () => {
  return (
    <ThirdwebProvider activeChain="ethereum">
      <YourApp />
    </ThirdwebProvider>
  );
};
```

### Adding Wallet Providers

To optionally add support for other wallets, such as [Gnosis Safe](/react/react.usegnosis), import the wallets to be included and add them to the `supportedWallets` prop.

```jsx
import {
  ThirdwebProvider,
  coinbaseWallet,
  metamaskWallet,
  safeWallet,
  walletConnectV1,
} from "@thirdweb-dev/react";

const App = () => {
  return (
    <ThirdwebProvider 
    // highlight-start
      supportedWallets={[
        metamaskWallet(),
        coinbaseWallet(),
        walletConnectV1(),
        safeWallet(),
      ]}
      // highlight-end
    activeChain="ethereum"
    >
      <YourApp />
    </ThirdwebProvider>
  );
};
```

With the provider set up, the connect button works out of the box with all your supported wallets! 

```jsx
import { ConnectWallet } from "@thirdweb-dev/react";

function App() {
  return <ConnectWallet />;
}
```

Wallets you provide to the [`ThirdwebProvider`'s `supportedWallets`](/react/react.thirdwebprovider#walletconnectors)
prop are shown as options
in the [`ConnectWallet`](/react/react.connectwallet) button&rsquo;s dropdown menu (if not configured, the default three options are shown).

### Using a Custom Connect Wallet Button

If you would like to use your own connect wallet button rather than using our UI component, simply use the `useConnect` hook:

```jsx
import { metaMask, useConnect } from "@thirdweb-dev/react";

function App() {
    const connect = useConnect(MetaMask(), {chainId=15})
    return (
        <div> 
            <button onClick={connect} />
        </div>
    )
}
```
