## Setting up a Wallet

To connect to a wallet, firstly create an instance of the wallet:

```typescript
const metamaskWallet = new MetaMaskWallet({
```

Secondly, connect to the wallet:

```typescript
try {
  const accountAddress = await metamaskWallet.connect();
  console.log("connected to account", accountAddress);
} catch (error) {
  console.log("Error connecting to wallet", error);
}
```

This will return a promise which will fulfill if the connection was successful and reject if the wallet was not connected.
