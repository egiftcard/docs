<h1 align="center">thirdweb Solidity SDK</h1>

Unified web3 Wallet library to integrate any wallet into your applications.

Build your own Connect Wallet button using our wallet components to create a custom wallet experience for your apps or games.

<br/>

:::note
Currently, Wallet SDK is only available for EVM environments
:::

## Supported Languages

- [React](/react)
- [React Native](/react-native)
- [Unity](/unity)
