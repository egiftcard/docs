Demonstration of all thirdweb's pre-built contracts with TypeScript + Getting started snippets

To create a new project using this template, use the [thirdweb CLI](/thirdweb-cli):

```jsx
npx thirdweb create --template contract-hub
```

## Using this template

This is an example repository showcases our most popular contracts, and a basic setup for interacting with them all.

**[Check out the Demo here](https://contract-hub.thirdweb-example.com)**!
