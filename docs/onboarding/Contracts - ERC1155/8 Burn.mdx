## Unlocked Features

By implementing the ERC721Mintable contract, you unlock the following features in the SDK and dashboard:

| SDK Usage                                                                                                | Description                                                                          |
| -------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------ |
| [Burn NFTs](/sdk/interacting-with-contracts/erc721/erc1155burnable#burn-a-specified-amount-of-a-nfts-1)  | Burn a specific quantity of NFTs                                                     |
| [Burn Batch of NFTs](/sdk/interacting-with-contracts/erc721/erc1155burnable#burn-a-batch-of-nfts)        | Burn a specific quantity of multiple NFTs in one transaction                         |
| [Burn From](/sdk/interacting-with-contracts/erc721/erc1155burnable#burn-a-specified-amount-of-a-nfts)    | Burn a specific quantity of NFTs from another wallet (requires permission/allowance) |
| [Burn Batch of NFTs From](/sdk/interacting-with-contracts/erc721/erc1155burnable#burn-a-batch-of-nfts-1) | Burn a specific quantity of NFTs                                                     |
