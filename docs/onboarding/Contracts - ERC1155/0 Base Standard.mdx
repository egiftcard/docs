# ERC1155

ERC1155 is the standard for representing "Edition" NFTs; where each NFT in the collection can have more than 1 quantity.

:::tip ERC1155 Starter Kit
For a full implementation of ERC1155 with several useful extensions, check out the [ERC1155Base Starter Kit](/extensions/base-contracts/erc-1155/erc1155base).
:::

## Unlocked Features

By implementing the ERC1155 standard, you unlock the following features in the SDK and dashboard:

| SDK Feature                                                                                    | Description                                                             |
| ---------------------------------------------------------------------------------------------- | ----------------------------------------------------------------------- |
| [Get NFT Balance](/sdk/interacting-with-contracts/erc1155/ERC1155#get-nft-balance)             | View the number of NFTs owned by a wallet address                       |
| [View NFT Metadata](/sdk/interacting-with-contracts/erc1155/ERC1155#get-a-single-nft-metadata) | View the token URI and fetch the metadata from IPFS                     |
| [Transfer](/sdk/interacting-with-contracts/erc1155/ERC1155#transfer-a-single-nft)              | Transfer this NFT from one wallet to another                            |
| [Airdrop](/sdk/interacting-with-contracts/erc721/erc1155#airdrop-multiple-nfts)                | Airdrop one or multiple NFTs to the provided a list of wallet addresses |
