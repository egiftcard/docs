Hook that allows users to connect their [Gnosis Safe](https://gnosis-safe.io/) wallet to your dApp.

```jsx
import { useSafe } from "@thirdweb-dev/react";
```

## Usage

First, configure your [`ThirdwebProvider`](/react/react.thirdwebprovider#supportedWallets)'s
`supportedWallets` prop to include the safeWallet:

```jsx
import { ThirdwebProvider } from "@thirdweb-dev/react";
import { safeWallet } from "@thirdweb-dev/react";

function MyApp() {
  return (
    <ThirdwebProvider
      activeChain={"ethereum"}
      supportedWallets={[safeWallet()]}
    >
      {/* Rest of your app here */}
      <YourApp />
    </ThirdwebProvider>
  );
}
```

Before connecting their Gnosis Safe wallet, user&rsquo;s must first:

1. Connect a personal wallet to your dApp (use [Connect Wallet Button](/react/react.connectwallet)).
2. Switch to the network the Gnosis Safe wallet is deployed to (use [`useSwitchChain`](/react/react.useSwitchChain)).

From this state, you are ready
to use the `useSafe` hook in your app:

```jsx
import { Goerli } from "@thirdweb-dev/chains";
import { useSafe } from "@thirdweb-dev/react";

const Home = () => {
  const connectWithGnosis = useSafe();

  return (
    <button
      onClick={() =>
        connectWithGnosis({
          safeAddress: "0x...", // Smart contract address of the Gnosis Safe wallet
          safeChainId: Goerli.chainId, // Chain ID the Gnosis Safe wallet is deployed to
        })
      }
    >
      Connect Gnosis Safe
    </button>
  );
};
```

Then use the [`useWallet`](/react/react.useWallet) hook to determine if the Gnosis wallet is connected:

```jsx
import { useWallet } from "@thirdweb-dev/react";

const Home = () => {
  const wallet = useWallet();
  const isSafeConnected = wallet && wallet.walletId === "Safe";
};
```

## Full Example

A [template repository](https://github.com/thirdweb-example/gnosis-safe)
is available for you to use, which includes the full setup for:

1. Connecting user&rsquo;s personal wallets
2. Enforcing the correct network connection
3. Connecting user&rsquo;s Gnosis Safe wallets
4. Displaying the user&rsquo;s connected wallet address
5. Disconnecting user&rsquo;s wallets
