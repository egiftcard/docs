Hook that prompts users to connect their Rainbow wallet to your app.

```jsx
import { useRainbowWallet } from "@thirdweb-dev/react";
```

The [rainbowWallet](/react/react.rainbowWallet) also needs to be added in [ThirdwebProvider's supportedWallets](/react/react.thirdwebprovider#supportedwallets-optional) if you want the wallet to auto-connect on next page load.

## Usage

Call the function returned by the `useRainbowWallet` hook to prompt the user to connect their Rainbow wallet to your dApp.

You can then use the [`useAddress`](/react/react.useaddress) hook to get the user's address.

```jsx
import { useRainbowWallet } from "@thirdweb-dev/react";

function App() {
  const connectWithRainbow = useRainbowWallet();

  return <button onClick={() => connectWithRainbow()}>Connect Rainbow</button>;
}
```

## Configuration

To connect to a specific chain when connecting the wallet,
pass the `chainId` in a configuration object as shown below.

This will prompt the user to switch to the given network after they connect.

```jsx
import { useRainbowWallet } from "@thirdweb-dev/react";
import { Dogechain } from "@thirdweb-dev/chains";

function App() {
  const connectWithRainbow = useRainbowWallet();

  return (
    <button
      onClick={() =>
        connectWithRainbow({
          chainId: Dogechain.chainId,
        })
      }
    >
      Connect Wallet
    </button>
  );
}
```

If the chain is not configured in the user&rsquo;s wallet,
you must add this chain in `ThirdwebProvider`&rsquo;s [supportedChains](/react/react.thirdwebprovider#supportedChains) prop:

```jsx
import { Dogechain } from "@thirdweb-dev/chains";
import { ThirdwebProvider } from "@thirdweb-dev/react";

export function YourApp() {
  return (
    <ThirdwebProvider supportedChains={[Dogechain]} clientId="your-client-id">
      <App />
    </ThirdwebProvider>
  );
}
```
