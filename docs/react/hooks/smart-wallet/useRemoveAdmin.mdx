Remove an admin on the smart wallet account. This action has to be performed by an admin on the account.

```jsx
const Component = () => {
  const { mutate: removeAdmin, isLoading, error } = useRemoveAdmin();

  if (error) {
    console.error("failed to remove admin", error);
  }

  return (
    <button disabled={isLoading} onClick={() => removeAdmin("0x...")}>
      Remove admin
    </button>
  );
};
```

## Usage

Provide the smart wallet account contract address as the argument.

```jsx
import { useAccountAdminsAndSigners } from "@thirdweb-dev/react";

// Your ERC20 token smart contract address
const accountAddress = "{{account_address}}";

const Component = () => {
  const { mutate: removeAdmin, isLoading, error } = useRemoveAdmin();

  if (error) {
    console.error("failed to remove admin", error);
  }

  return (
    <button disabled={isLoading} onClick={() => removeAdmin(accountAddress)}>
      Remove admin
    </button>
  );
};
```
