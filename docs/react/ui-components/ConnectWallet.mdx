Button to connect a user&rsquo;s wallet to your app, with support for
[MetaMask](/react/react.usemetamask),
[Coinbase](/react/react.usecoinbasewallet), and
[WalletConnect](/react/react.usewalletconnect).

Wallets you provide to the [`ThirdwebProvider`'s walletConnectors](/react/react.thirdwebprovider#walletconnectors)
prop are shown as options
in the button&rsquo;s dropdown menu (if not configured, the default three options are shown).

```jsx
import { ConnectWallet } from "@thirdweb-dev/react";
```

## Usage

Render the `ConnectWallet` component to display the button.

```jsx
import { ConnectWallet } from "@thirdweb-dev/react";

function App() {
  return <ConnectWallet />;
}
```

## Configuration

### accentColor (optional)

The background color of the button.

Can be a color in the form of a hex code, a CSS color name, `transparent`, or `initial`.

The default value is `#000000`.

```jsx
import { ConnectWallet } from "@thirdweb-dev/react";

function App() {
  return (
    <ConnectWallet
      // highlight-next-line
      accentColor="pink"
    />
  );
}
```

### colorMode (optional)

Change the color scheme of the button to `light` or `dark` mode, to match the color scheme of your app.

The default value is `dark`.

```jsx
import { ConnectWallet } from "@thirdweb-dev/react";

function App() {
  return (
    <ConnectWallet
      // highlight-next-line
      colorMode="light"
    />
  );
}
```

### btnTitle (optional)

Change the text the button displays while in the disconnected state.

The default value is `Connect Wallet`.

```jsx
import { ConnectWallet } from "@thirdweb-dev/react";

function App() {
  return (
    <ConnectWallet
      // highlight-next-line
      btnTitle="Connect Wallet"
    />
  );
}
```

### auth (optional)

Enforce that users must sign in with their wallet using [auth](/auth) after connecting their wallet.

Requires the [`authConfig`](/react/react.thirdwebprovider#authconfig) prop to be set on the [`ThirdWebProvider`](/react/react.thirdwebprovider) component.

```jsx
import { ConnectWallet } from "@thirdweb-dev/react";

function App() {
  return (
    <ConnectWallet
      colorMode="light"
      auth={{
        loginOptional: false,
      }}
    />
  );
}
```

### className (optional)

Apply custom CSS styles to the button.

:::info

For some specific CSS attributes such as `width` you may need to apply the
[`!important`](https://developer.mozilla.org/en-US/docs/Web/CSS/Specificity#the_!important_exception)
CSS rule to override the default styles of the button.
:::

```jsx
import { ConnectWallet } from "@thirdweb-dev/react";

function App() {
  return (
    <ConnectWallet
      // highlight-next-line
      className="my-custom-class"
    />
  );
}
```
