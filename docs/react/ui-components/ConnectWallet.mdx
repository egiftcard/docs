Button to connect a user&rsquo;s wallet to your app, with support for MetaMask, Coinbase, WalletConnect, Safe Wallet and Paper Wallet.

Wallets you provide to the [`ThirdwebProvider`'s supportedWallets](/react/react.thirdwebprovider#supportedWallets)
prop are shown as options in the ConnectWallet button&rsquo;s Modal.
If this prop is not provided, the default values are shown (MetaMask, Coinbase Wallet, WalletConnect).

```jsx
import { ConnectWallet } from "@thirdweb-dev/react";
```

## Usage

Render the `ConnectWallet` component to display the button.

```jsx
import { ConnectWallet } from "@thirdweb-dev/react";

function App() {
  return <ConnectWallet />;
}
```

## Configuration

### modalSize (optional)

ConnectWallet modal has two sizes - "wide" and "compact"

The default is `"wide"` for desktop and `"compact"` for mobile devices

ConnectWallet Modal is always `"compact"` on mobile and the `modalSize` prop is ignored.

### theme (optional)

Change the theme of Connect Wallet UI to `"light"` or `"dark"` mode, to match the color theme of your app.

The default value is `"dark"`.

```jsx
import { ConnectWallet } from "@thirdweb-dev/react";

function App() {
  return (
    <ConnectWallet
      // highlight-next-line
      theme="dark"
    />
  );
}
```

You can also create a custom theme by passing an object. To do this, you can use `darkTheme`
or `lightTheme` functions to use light / dark theme as base and override it

```tsx
import { darkTheme, lightTheme } from "@thirdweb-dev/react";

<ConnectWallet
  // highlight-next-line
  theme={darkTheme({
    fontFamily: "Inter, sans-serif",
    colors: {
      modalBg: "#000000",
      accentText: "red",
      // ... etc
    },
  })}
/>;
```

### btnTitle (optional)

Change the text the button displays while in the disconnected state.

The default value is `"Connect Wallet"`.

```jsx
import { ConnectWallet } from "@thirdweb-dev/react";

function App() {
  return (
    <ConnectWallet
      // highlight-next-line
      btnTitle="Connect Wallet"
    />
  );
}
```

### Customize "Welcome Screen" for wide modal

On wide modal, a welcome screen is shown on the right side of the modal.

This screen can be customized in two ways

#### 1. customize metadata

```tsx
<ConnectWallet
  welcomeScreen={{
    title: "Your Title",
    subtitle: "Your Subtitle",
    img: {
      src: "https://...",
      width: 300,
      height: 50,
    },
  }}
/>
```

#### 2. Render custom component

```tsx
<ConnectWallet
  welcomeScreen={() => {
    return <YourComponent />;
  }}
/>
```

### modalTitle (optional)

Change the title of Connect Wallet Modal's title

The default is `"Connect"`.

```jsx
import { ConnectWallet } from "@thirdweb-dev/react";

function App() {
  return (
    <ConnectWallet
      // highlight-next-line
      modalTitle="Login"
    />
  );
}
```

### detailsBtn (optional)

Render a custom button to display connected wallet details

```jsx
import { ConnectWallet } from "@thirdweb-dev/react";

function App() {
  return (
    <ConnectWallet
      // highlight-start
      detailsBtn={() => {
        return <button> .... </button>;
      }}
      // highlight-end
    />
  );
}
```

### auth (optional)

Enforce that users must sign in with their wallet using [auth](/auth) after connecting their wallet.

Requires the [`authConfig`](/react/react.thirdwebprovider#authconfig) prop to be set on the [`ThirdWebProvider`](/react/react.thirdwebprovider) component.

```jsx
import { ConnectWallet } from "@thirdweb-dev/react";

function App() {
  return (
    <ConnectWallet
      // highlight-start
      auth={{
        loginOptional: false,
      }}
      // highlight-end
    />
  );
}
```

### className (optional)

Apply custom CSS styles to the button.

:::info

For some specific CSS properties, you may need to apply the
[`!important`](https://developer.mozilla.org/en-US/docs/Web/CSS/Specificity#the_!important_exception)
CSS rule to override the default styles

:::

```jsx
import { ConnectWallet } from "@thirdweb-dev/react";

function App() {
  return (
    <ConnectWallet
      // highlight-next-line
      className="my-custom-class"
    />
  );
}
```

### switchToActiveChain (optional)

Specify whether to show a "Switch Network" button after the wallet is connected but it is not connected to the [activeChain](/react/react.thirdwebprovider#activechain-recommended) set in [ThirdwebProvider](react/react.thirdwebprovider) to encourage the user to switch to the correct network in their wallet.

[activeChain](/react/react.thirdwebprovider#activechain-recommended) must be explicitly set in [ThirdwebProvider](react/react.thirdwebprovider) for this to work.

By default, it is set to `false`.

```jsx
<ConnectWallet switchToActiveChain={true} />
```

### dropdownPosition (optional)

Specify where should the dropdown menu open relative to the Connect Wallet Button.

```jsx
<ConnectWallet
  dropdownPosition={{
    side: "bottom", // "top" | "bottom" | "left" | "right";
    align: "end", // "start" | "center" | "end";
  }}
/>
```

### hideTestnetFaucet (optional)

Hide option to request testnet funds for testnets in dropdown

```jsx
<ConnectWallet hideTestnetFaucet={true} />
```

### Show "Terms of Service" and/or "Privacy Policy" links

You can show a "Terms of Service" and/or "Privacy Policy" link in the ConnectWallet Modal by passing the `termsOfServiceUrl` and/or `privacyPolicyUrl` props

```tsx
<ConnectWallet
  termsOfServiceUrl="https://...."
  privacyPolicyUrl="https://...."
/>
```

## supportedTokens (optional)

Customize the tokens shown in the "Send Funds" screen for various networks.

By default, The "Send Funds" screen shows a few popular tokens on for default chains and the native token. For other chains it only shows the native token.

<img src="/assets/wallets/select-token-dark.png" />

<br />
<br />

`supportedTokens` prop allows you to customize this list as shown below

```tsx
import { Base } from "@thirdweb-dev/chains";

<ConnectWallet
  supportedTokens={{
    // Show "Dai Stablecoin" when connected to the "Base" mainnet
    [Base.chainId]: [
      {
        address: "0x50c5725949A6F0c72E6C4a641F24049A917DB0Cb",
        name: "Dai Stablecoin",
        symbol: "DAI",
        icon: "https://assets.coingecko.com/coins/images/9956/small/Badge_Dai.png?1687143508",
      },
      // ...etc
    ],
  }}
/>;
```

### displayBalanceToken

Display the balance of a token instead of the native token in ConnectWallet details button.

```tsx
<ConnectWallet
  displayBalanceToken={{
    // show Wrapped BTC balance if connected to Ethereum mainnet
    1: "0x2260FAC5E5542a773Aa44fBCfeDf7C193bc2C599",
  }}
/>
```
