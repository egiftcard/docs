Hook for signing out of a wallet after a user has [logged in](/react/react.uselogin) using [auth](auth).

```jsx
import { useLogout } from "@thirdweb-dev/react";
```

## Usage

Call the `login` function to have the user sign out.

The `isLoading` boolean can be used to display a loading state on the UI while the user is signing out.

```jsx
import { useLogout } from "@thirdweb-dev/react";

function App() {
  const { logout, isLoading } = useLogout();

  return (
    <button onClick={() => logout()}>
      {isLoading ? "Logging out..." : "Logout"}
    </button>
  );
}
```
