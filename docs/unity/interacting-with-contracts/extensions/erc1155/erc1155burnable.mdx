# ERC1155Burnable

Functionality available for contracts that implement the
[`IERC1155Enumerable`](/contracts/IERC1155Enumerable) and
[`IBurnableERC1155`](/contracts/IBurnableERC1155)
interfaces.

Allows you to burn NFTs (transfer them to a null address) to take them out of circulation.

## burnBatch

Burn multiple different NFTs in the same transaction from the **connected wallet**.

```csharp

```

<details>
  <summary>Configuration</summary>
  <div>

#### tokenIds

An array of token IDs of the NFTs you want to burn.

Must be an array of `string`, `number`, or `BigNumber`.

```csharp

```

#### amounts

An array of the amounts of each NFT you want to burn.

The index of each amount corresponds to the index of the token ID in the `tokenIds` array.

For example, the first item in the `tokenIds` array will have the first item in the `amounts` array burned.

Must be an array of `string`, `number`, or `BigNumber`.
