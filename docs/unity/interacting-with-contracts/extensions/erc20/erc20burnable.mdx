# ERC20Burnable

Functionality available for contracts that implement the
[`IERC20`](/contracts/IERC20) and
[`IBurnableERC20`](/contracts/IBurnableERC20) interfaces.

Allows you to burn tokens (take them out of circulation).

## Burn

Burn tokens held by the **connected wallet**.

```csharp
var data = await contract.ERC20.Burn("{{amount}}");
```

<details>
  <summary>Configuration</summary>
  <div>

#### amount

The amount of this token you want to burn.

Must be a `string`.

</div>
</details>
