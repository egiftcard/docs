# ERC20Mintable

Functionality available for contracts that implement the
[`IERC20`](/contracts/IERC20) and
[`IMintableERC20`](/contracts/IMintableERC20) interfaces.

Allows the minting of new tokens into the contract.

## Mint

Mint tokens to the **connected wallet**.

```csharp
var data = await contract.ERC20.Mint("{{amount}}");
```

<details>
  <summary>Configuration</summary>
  <div>

#### amount

The amount of tokens to mint.

Must be a `string`.

</div>
</details>

## MintTo

The same as [`mint`](#mint), but allows you to specify the address to mint the tokens to.

```csharp
var data = await contract.ERC20.MintTo("{{wallet_address}}", "{{amount}}");
```

<details>
  <summary>Configuration</summary>
  <div>

#### to

The wallet address to mint the tokens to.

Must be a `string`.

#### amount

The number of tokens to mint.

Must be a `string`.

</div>
</details>
