# ERC721Enumerable

## getOwned

Get all the data associated with the NFTs owned by a specific wallet.

```javascript
// Address of the wallet to get the NFTs of
const address = "{{wallet_address}}";
const nfts = await contract.erc721.getOwned(address);
console.log(nfts);
```
