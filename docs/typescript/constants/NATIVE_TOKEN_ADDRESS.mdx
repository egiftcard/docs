The address of the native token to the current chain.

For example, the native token on Ethereum is Ether, for Polygon it is Matic, etc.

## Usage

```js
import { NATIVE_TOKEN_ADDRESS } from "@thirdweb-dev/sdk";
```

## Return Value

The native token address is the [null address](https://polygonscan.com/address/0xeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeeee).

```typescript
"0xEeeeeEeeeEeEeeEeEeEeeEEEeeeeEeeeeeeeEEeE";
```
