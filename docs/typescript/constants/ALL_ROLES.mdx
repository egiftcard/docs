An array containing all the built-in roles supported by the SDK used for [Permissions](/typescript/sdk.permissions).

## Usage

Contains all the supported roles that can be used with the `sdk.role` methods.

```javascript
import { ALL_ROLES } from "@thirdweb-dev/sdk";
```

## Return Value

Returns an array of strings with all the supported roles.

```typescript
[
  "admin",
  "transfer",
  "minter",
  "pauser",
  "lister",
  "asset",
  "unwrap",
  "factory",
];
```
