When using the [Pack](https://thirdweb.com/thirdweb.eth/Pack) or [Pack (Chainlink VRF)](https://thirdweb.com/thirdweb.eth/PackVRFDirect)
smart contract, additional top-level functionality is available to use.

## Get a Pack

```javascript
const contract = await sdk.getContract("{{contract_address}}", "pack");
```
