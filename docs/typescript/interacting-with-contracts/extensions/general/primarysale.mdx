# PrimarySale

Functionality available for contracts that inherit the [`PrimarySale`](/contracts/PrimarySale) contract.

## getRecipient

Get the primary sale recipient.

```javascript
const salesRecipient = await contract.sales.getRecipient();
```

<details>
    <summary>Configuration</summary>
    <div>

### Return Value

Returns a `string` containing the wallet address of the primary sale recipient.

```typescript
string;
```

</div>
</details>

## setRecipient

Set the primary sale recipient.

```javascript
await contract.sales.setRecipient("{{wallet_address}}");
```

<details>
    <summary>Configuration</summary>
    <div>

#### recipient

The wallet address of the primary sale recipient.

Must be a `string`.

</div>
</details>
