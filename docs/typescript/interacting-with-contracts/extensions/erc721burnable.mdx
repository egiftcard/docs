# ERC721Burnable

Functionality available for contracts that implement the
[`IERC721`](/contracts/IERC721) and
[`IBurnableERC721`](/contracts/IBurnableERC721)
interfaces.

## burn

Burn a single NFT

```javascript
const result = await contract.erc721.burn(tokenId);
```
