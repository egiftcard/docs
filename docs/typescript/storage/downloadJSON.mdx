Download JSON data from any URL or IPFS URI.

## Usage

Pass a URL (or IPFS URI) to this method to download the JSON data.

```jsx
const json = await sdk.storage.downloadJSON("ipfs://example");
```

## Configuration

### url

The URL of the JSON data to download.

Must be a `string`.

## Return Value

Returns the JSON data fetched from the resolved URL as `any`.

If there is no data found at the URL provided, returns `undefined`.

```typescript
any | undefined;
```
