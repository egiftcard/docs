Remove all listeners form a contract, such as with [`addEventListener`](/typescript/sdk.contractevents.addeventlistener),
or [`addTransactionListener`](/typescript/sdk.contractevents.addtransactionlistener).

## Usage

To remove all listeners from a contract, call the `removeAllListeners` method.

```javascript
contract.events.removeAllListeners();
```
